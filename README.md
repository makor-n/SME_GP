# SME_GP Dashboard

แดชบอร์ดวิเคราะห์การกระจายตัวของ SME ทั่วประเทศไทย

## ข้อมูล
- **แหล่งข้อมูล:** SME_GP_ALL_230369 (สสว.)
- **จำนวนกิจการ:** 140,768 ราย
- **ครอบคลุม:** 77 จังหวัด · 999 ประเภทอุตสาหกรรม

## ฟีเจอร์
- 🗺️ **แผนที่** — Bubble Map แสดงการกระจายตัว SME รายจังหวัด สีแยกตามภาค
- 🔍 **ค้นหา & กรอง** — เลือกจังหวัด → ติ๊กอุตสาหกรรมหลายรายการ → ดูรายชื่อบริษัท เบอร์โทร อีเมล
- 📊 **สรุปรายจังหวัด** — Micro / Small / Medium / นิติบุคคล / บุคคลธรรมดา / วิสาหกิจชุมชน
- ♾️ **Infinite Scroll** — โหลดข้อมูลทีละ 100 รายการ ไม่กระตุก

## วิธีใช้
เปิดไฟล์ `index.html` ในเบราว์เซอร์ หรือเปิดผ่าน GitHub Pages

## GitHub Pages
เปิด **Settings → Pages → Source: main / root** แล้วเข้าได้ที่:
```
https://makor-n.github.io/SME_GP/
```
